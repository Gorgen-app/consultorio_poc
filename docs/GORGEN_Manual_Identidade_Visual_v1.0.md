## 1. INTRODUÇÃO E FILOSOFIA DA MARCA

### 1.1 Sobre Este Manual

Este Manual de Identidade Visual é um documento normativo que estabelece todas as regras, especificações técnicas e diretrizes para a aplicação correta da identidade visual do GORGEN. O cumprimento rigoroso destas normas é obrigatório para garantir a consistência, reconhecimento e integridade da marca.

### 1.2 Missão do GORGEN

> *"Somos uma plataforma de gestão em saúde com foco no paciente, no qual médicos são parte do ecossistema que desenvolve toda a jornada assistencial — da consulta ao faturamento, do intra- ao extra-hospitalar. Ouvimos as dores de pacientes e de médicos diante de um sistema de saúde burocrático, fragmentado e distorcido e desenhamos uma solução inteligente que simplifica a gestão e devolve tempo e energia dos médicos para o que realmente importa: o cuidado às pessoas."*

### 1.3 Valores da Marca

| Valor | Descrição |
|-------|-----------|
| **Confiança** | Segurança e sigilo absoluto dos dados de saúde |
| **Clareza** | Interface intuitiva e comunicação transparente |
| **Inovação** | Tecnologia a serviço do cuidado humano |
| **Humanização** | O paciente no centro de todas as decisões |

### 1.4 Personalidade Visual

A identidade visual do GORGEN foi concebida para transmitir:

- **Profissionalismo médico** sem frieza institucional
- **Tecnologia avançada** com acessibilidade humana
- **Segurança e confiabilidade** através de cores sóbrias
- **Modernidade** sem modismos passageiros

O logotipo do GORGEN é representado por um **farol** — símbolo universal de:

| Significado | Interpretação no Contexto GORGEN |

```

│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒ │

│ ▒▒    │                  │       ▒▒ │
│ ▒▒    │     LOGOTIPO     │       ▒▒ │
│ ▒▒    │                  │       ▒▒ │
│ ▒▒    └──────────────────┘       ▒▒ │
│ ▒▒                                ▒▒ │
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒ │
└────────────────────────────────────────┘

▒▒ = Área de proteção (10% do diâmetro)
```

**Cálculo:** Se o logotipo tem 500px de diâmetro, a área de proteção é de 50px em todas as direções.

**PÁGINA 8**

### 2.6 Proporções Oficiais

As proporções do logotipo são **IMUTÁVEIS**. O logotipo **NUNCA** deve ser:

- Esticado horizontalmente
- Esticado verticalmente
- Distorcido de qualquer forma
- Rotacionado (exceto em casos específicos autorizados)

#### 2.6.1 Proporção Farol/Círculo

A proporção do farol em relação ao círculo de fundo foi definida com base no arquivo master oficial:

| Elemento | Proporção em relação ao diâmetro |
|----------|----------------------------------|
| **Altura do farol (base ao topo)** | ~70% do diâmetro |
| **Largura do farol (base)** | ~35% do diâmetro |
| **Margem superior** | ~8% do diâmetro |
| **Margem inferior (ondas)** | ~12% do diâmetro |
| **Raios de luz** | Estendem-se até ~45% do diâmetro a partir do centro |

### 2.7 Nomenclatura de Arquivos

Todos os arquivos de logotipo **DEVEM** seguir o padrão de nomenclatura:

```
gorgen_logo_[versão]_[dimensão]_[variante].[extensão]
```

**Exemplos:**
- `gorgen_logo_master_2048_black.png`
- `gorgen_logo_white_bg_500x500.png`
- `gorgen_logo_master_official.svg`

### 4.4 Fonte Alternativa (Fallback)

```css

             -apple-system,

             'Segoe UI', 
             Roboto, 
             'Helvetica Neue', 
             Arial, 
             sans-serif;
```

### 4.6 Regras de Uso Tipográfico

| Regra | Descrição |
|-------|-----------|
| **Alinhamento** | Textos longos sempre alinhados à esquerda |
| **Largura máxima** | 70-80 caracteres por linha para leitura confortável |
| **Contraste** | Mínimo 4.5:1 para texto normal, 3:1 para texto grande |
| **Hierarquia** | Máximo 3 níveis de hierarquia por tela |
| **Maiúsculas** | Usar apenas em labels curtos e botões quando apropriado |
