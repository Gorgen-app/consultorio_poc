## Instruções de Uso

### 1. Escolha a Versão Correta

- **Dark**: Use quando o contexto visual da plataforma for predominantemente claro
- **Light**: Use quando o contexto visual da plataforma for predominantemente escuro

### 2. Não Modifique os Arquivos

Os arquivos foram gerados seguindo as especificações do Manual de Identidade Visual. Qualquer modificação requer aprovação do Administrador Master.

### 3. Mantenha a Consistência

Use a mesma versão (dark ou light) em todas as plataformas para manter a consistência da marca.
